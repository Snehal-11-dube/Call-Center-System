# Call-Center-System
 A call center receives incoming calls, and each call is assigned a unique customer ID. The  calls are answered in the order they are received. 
1. addCall(customerID,callTime): Add a call to the queue with the customerID and the calltime(in minutes).
2. answerCall(): Answer and remove the first call from the queue.
3. viewQueue():View all calls currently in the queue without removing them.
4. isQueueEmpty(): Check if the queue is empty.
